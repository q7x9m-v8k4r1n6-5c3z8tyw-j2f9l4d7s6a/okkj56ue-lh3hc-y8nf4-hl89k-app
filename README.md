# OVC-MOVE-APP

## Giới thiệu dự án

OVC-MOVE-APP là ứng dụng frontend của hệ thống tính điểm, hỗ trợ vận hành vòng RACE trong chương trình MOVE.

Production: [https://move.oispvolunteerclub.com/](https://move.oispvolunteerclub.com/)
